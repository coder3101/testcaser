Detailed Description

template<class Generator = std::mt19937, class Distribution = std::uniform_int_distribution<long long>>
class testcaser::maker::types::RandomInteger< Generator, Distribution >

RandomInteger object generates a random integer every time .get() is called it follows the generator and the distribution type that you provide.
