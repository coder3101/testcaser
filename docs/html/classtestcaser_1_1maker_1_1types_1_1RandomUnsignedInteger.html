Detailed Description

template<class Generator = std::mt19937, class Distribution = std::uniform_int_distribution<unsigned long long>>
class testcaser::maker::types::RandomUnsignedInteger< Generator, Distribution >

RandomUnsignedInteger object generates a random integer every time .get() is called it follows the generator and the distribution type that you provide.
