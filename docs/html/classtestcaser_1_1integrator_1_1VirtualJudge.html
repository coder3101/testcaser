Detailed Description

The VirtualJudge Class use to create a new virtual judge session to run a program.

Constructor & Destructor Documentation

◆ VirtualJudge()

testcaser::integrator::VirtualJudge::VirtualJudge ( )
inline

Construct a new VirtualJudge object. Default time limit is 1 sec, 30s is auto exit time when auto exit is false. If program does not exists in 30s although it has passed its time limit, It will recieve a SIGKILL. It also defaults 256 MB of memory limit. If not explicitly set by the programmer.
