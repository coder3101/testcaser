Detailed Description

template<class Generator = std::mt19937, class Distribution = std::uniform_int_distribution<unsigned long long>>
class testcaser::maker::types::RandomAlphabet< Generator, Distribution >

RandomAlphabet object generates a random alphabet every time .get() is called it follows the generator and the distribution type that you provide.

Template Parameters
